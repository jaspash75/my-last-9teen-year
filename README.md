## 🌸 About This Project

This repository is a personal time capsule that celebrates my **19th birthday** and the beginning of my final year as a teenager.

Rather than being just another Python program, it's a snapshot of who I am at 19—my mindset, dreams, aspirations, and excitement for everything that's ahead.

I hope that one day, whether it's my 20th birthday or many years later, I can come back to this project and remember where my journey truly began.

---

##  🎂What This Project Demonstrates

This project is written in beginner-friendly Python and showcases fundamental programming concepts, including:

- Variables
- Strings
- Arithmetic, Comparison & Logical Operators
- Input & Output
- Conditional Statements (`if`, `else`, nested `if`)
- `match-case` (Python's switch-case)
- `for` and `while` loops
- Lists
- Dictionaries
- Functions
- Formatted Strings (f-strings)
- Working with Dates using the `datetime` module
